Workflow skills name capabilities, not tools.

--- skills/respond-to-review/SKILL.md
name: respond-to-review
description: Address review feedback on your own merge request
metadata:
  requires:
    - source-control
    - tdd
    - push
    - verify
---

# Body: imperative steps. Names capabilities, never tools.
The body never names a tool. Switch from one source-control provider to another and the workflow doesn't change a line.

Where tools attach: capabilities.yaml.

# pure CLI — when --help is enough
meetings: cli://minutes
email:    cli://gws

# CLI with one or two gotchas — note inline, no skill file
reminders:
  provider: cli://remindctl
  notes: >-
    `show upcoming` only returns reminders with a due date —
    use `show all` to include undated items.

# external skill, maintained upstream by the CLI author
calendar: skill://ical-cli   # installed via gh skill install

# local integration skill — only when gotchas don't fit in `notes`
chat:     skill://slack
CLI when --help is enough; skill when it isn't. Workflow skills point at capability names; this file points capability names at providers. Swap the right side without touching the left.

Shift left: the plan is read-only deliberation; every remote write asks for approval. Course-correction is cheap before code, expensive after.

attention — energy in, output calibrated.

Low
One work item + one personal. Quick wins or rest.
Medium
Two or three short-cycle items. Reviews, fixes, follow-ups.
High
Longer-horizon work. New features, architectural moves, deep reads.

Who shapes the workflow itself?

Files in git · chezmoi
Skills as files
Every modification a commit. Reproducible across machines via chezmoi apply. Auditable, portable, diff-able.
Cloud memory · OpenClaw & co.
Skills as agent state
Chat-native, proactive, self-modifying. Faster to bootstrap. Modifications happen in territory the runtime owns, without commits.
Either stance, same ceiling
Two or three streams is the cap — autonomous or supervised, the human bottleneck holds. Shift left makes each loop cheaper. It does not move the ceiling.
