---
description: Create a weekly event digest for local events, sports, and streaming
---

Create a weekly event digest. Search for and compile information from:

## Local Events

Search for events in the configured local area:
- Theatre and performing arts shows
- Community events and family activities
- Library events and programs

## Sports This Week

- Notable NFL games (matchups with playoff implications, rivalries)
- Notable NBA games (top teams, rivalries)
- Filter for games on broadcast/streaming services available to the user

## New TV Shows & Streaming

- New shows premiering this week on major streaming services
- New seasons of popular shows returning

## Configuration

This command uses local overrides for personalization. Create `~/.config/opencode/command.local/event-digest.md` with:
- Your local area/towns
- Specific venues and theatres
- TV services you have access to
- Family-friendly preferences

## Output

Format the digest as a clean, scannable summary organized by category. Include dates, times, and where to watch/attend.
