# Zwicky Box

**Category:** Problem Solving
**Also called:** Morphological Box
**Source:** Fritz Zwicky (Swiss astronomer, 20th century)

Use this when standard brainstorming feels exhausted, the problem has multiple independent dimensions, or you need to explore a large solution space systematically rather than intuitively.

## Core Principles

Two principles of creativity underlie this method:
1. **Decomposition:** Break the problem into independent attributes.
2. **Forced association:** Combine values from different attributes to discover non-obvious solutions.

## Steps

### 1. Define the problem clearly
What are you trying to solve? Be specific.

### 2. Identify independent attributes
Decompose the problem into its key dimensions — aspects, components, or properties that can vary independently.

**Tip:** Look at an existing solution and reverse-engineer its components. Each component is a candidate attribute.

Limit to 4–7 attributes for tractability.

### 3. Fill in values for each attribute
For each attribute, list 3–6 possible values. Don't filter — include unconventional or unusual options. Quantity matters here.

### 4. Generate combinations
Pick one value from each attribute column to form a complete solution concept. Do this:
- Randomly (to force unexpected combinations)
- Systematically (to cover the space)
- Focused (fix 1–2 interesting values and vary the rest)

Evaluate each combination: does it suggest something worth developing?

## Agent Workflow

When asked to apply the Zwicky Box:

1. Confirm the problem statement.
2. Propose 4–6 attributes. Discuss and refine with the user.
3. Generate 3–5 values per attribute.
4. Produce a formatted table (the Zwicky Box).
5. Generate 3–5 combinations, describe each as a concrete solution concept.
6. Flag the 1–2 most promising combinations for further development.

## Example

**Problem:** Improve how we collect customer feedback.

| Collection Method | Timing | Format | Incentive | Analysis |
|------------------|--------|--------|-----------|----------|
| Email survey | On cancellation | Open text | Gift card | AI sentiment |
| In-app popup | After key action | Multiple choice | Discount | Manual review |
| Phone interview | Weekly | Audio | Early access | Trend analysis |
| Social listening | During usage | Video | Public recognition | Team discussion |

**Combination A:** In-app popup + After key action + Multiple choice + Discount + AI sentiment
→ Triggered micro-surveys after key actions, offering a discount for completion, with automated sentiment tracking.

**Combination B:** Phone interview + Weekly + Audio + Early access + Team discussion
→ Weekly async voice message drop from power users, rewarded with beta access, discussed in team sync.

**Combination C (forced/unusual):** Social listening + During usage + Video + Public recognition + Trend analysis
→ Monitor social when users are actively using the app, surface video responses publicly and track trends over time.

## Tips

- The goal is **volume of combinations** — most won't be good, but some will be unexpectedly strong.
- Don't dismiss combinations that seem strange at first. Sit with them.
- Works best with 3–5 people who can quickly evaluate combinations.
